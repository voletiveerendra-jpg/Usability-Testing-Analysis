# Usability Testing & Analysis

## Overview
This project focuses on conducting usability testing for a food delivery mobile application prototype. The objective was to evaluate user interactions, identify usability issues, and suggest improvements for a better user experience.

## Objectives
- Create usability test plans and scenarios
- Conduct testing sessions with users
- Collect and analyze feedback
- Identify usability problems
- Recommend design improvements

## Test Scenarios
1. Create a new account
2. Search for a restaurant
3. Add food items to the cart
4. Complete the checkout process
5. Track an order

## Participants
- 5 Test Users
- Age Group: 18–35 years

## Findings
- Users struggled to locate the search feature.
- Checkout button visibility was low.
- Order tracking page lacked clear status indicators.

## Recommendations
- Improve search bar visibility.
- Increase checkout button size and contrast.
- Add progress indicators for order tracking.

## Tools Used
- Figma
- Google Forms
- Microsoft Excel

## Outcome
The project demonstrated how usability testing helps improve user satisfaction and product effectiveness.1
